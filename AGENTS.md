# Visit Colombia - Agents Manifesto

## Mission
To be the most comprehensive, beautiful, and accessible resource for travelers visiting Colombia. This project provides a statically generated Next.js website deployed on GitHub Pages.

## Core Capabilities
Everything you need to work on this project is organized into **Skills**.

- **[api-client](./skills/api-client/SKILL.md)**: interact with the Open Data Colombia API.
- **[content-management](./skills/content-management/SKILL.md)**: Manage blog posts and static content.
- **[deployment](./skills/deployment/SKILL.md)**: Build and deploy the application.

## Development Guidelines
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS
- **Deployment**: GitHub Pages (Static Export)
- **Language**: TypeScript

## Quick Start
1. **Install**: `npm install`
2. **Dev**: `npm run dev`
3. **Build**: `npm run build`
4. **Serve**: `npx serve@latest out`
